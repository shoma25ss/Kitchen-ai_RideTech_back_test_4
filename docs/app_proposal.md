## 2. 開発理由と目的

### 開発理由

自分も料理をする中で、2つの大きなストレスを感じていました。

1つ目は、仕事終わりに「今日何を作ろう？」と考えること自体が、疲れた体にとって大きな負担になること。

2つ目は、SNSで見つけたレシピを作る際に、動画を止めながら文章を確認するという行き来が非常にストレスで、余計な時間と手間がかかってしまうことです。

この2つの課題を同時に解決するアプリをAIの力を使って開発しようと考えました。

### 目的

冷蔵庫にある食材を入力するだけでレシピが自動生成され、そのまま手順ナビ＆タイマーで調理を完結できる体験を提供すること。料理の「決める・調べる・作る」すべてのストレスをゼロにする。

### 想定ユーザー

一人暮らしの大学生・社会人、忙しい共働き夫婦、料理初心者全般

---

## 3. 背景

### 献立の悩みは毎日発生する

「今日何を作ろう？」と悩む時間は積み重なると大きなストレスになります。
特に一人暮らしや共働き世帯では、料理に使える時間・エネルギーが限られています。

### 食材の無駄が社会課題になっている

冷蔵庫に食材があるのに使い切れず捨ててしまうケースは非常に多いです。
「今ある食材から作れるレシピ」を即座に提案することで食品ロスを減らせます。

### 既存サービスとの差別化

クックパッド等は「食べたいものを検索する」体験であり、食材起点でレシピを自動生成し、そのまま調理ナビまで完結するサービスは少ないです。AIを活用することでこのギャップを埋められます。

## 4. アプリができることの説明

| 機能名 | できること | 詳細 |
|------|------|------|
| ユーザー登録・ログイン | アカウントを作成してログインできる | 会員登録・ログイン・パスワードリセット機能を実装。ユーザーごとにお気に入りレシピや料理履歴を管理できる。 |
| AIレシピ自動生成 | 食材を入力するだけでレシピを自動作成 | 食材名を入力するとAIが料理名・材料・手順・調理時間・カロリーを自動生成。3〜5品の候補から好みの料理を選択できる。 |
| フィルター絞り込み | 条件を指定して料理を絞り込む | ジャンル（和・洋・中）、調理時間（15分以内など）、カロリー、難易度でフィルタリングが可能。 |
| 手順ナビ機能 | 料理の手順を1ステップずつ表示 | 生成されたレシピの手順を1つずつ画面に表示。初心者でも迷わず調理を進められる。 |
| タイマー連動 | 各手順に合わせてタイマーが自動セット | 「中火で5分炒める」などの手順を検知し、タイマーが自動でカウントダウンを開始する。 |
| AIチャット質問 | 調理中に疑問をAIに質問できる | 「代わりの食材は？」「火加減は？」など調理中の疑問をチャットでAIに質問し、即座に回答を得られる。 |
| 料理履歴 | 作った料理の履歴を確認できる | 過去に作った料理を一覧で確認できる。日付・料理名・使った食材を記録し振り返ることができる。 |
| みんなのレシピ | 他のユーザーが作ったレシピを一覧できる | 他のユーザーがAIで生成したレシピを一覧表示。参考にしたいレシピをそのまま自分の調理ナビで使える。 |
| よく使う食材の保存 | よく使う食材を登録しておける | よく使う食材をあらかじめ登録しておくことで、毎回入力する手間を省ける。 |
