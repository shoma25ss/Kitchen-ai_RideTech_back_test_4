# KitchenAI 要件定義書

## アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | KitchenAI |
| 概要 | 食材を入力するとAIがレシピを自動生成し、手順ナビ＆タイマーで調理を完結できるアプリ |
| 技術スタック | HTML / CSS / JavaScript・Laravel・MySQL・Docker・Claude API |

---

## ユーザーロール

| ロール名 | 権限概要 |
|----------|----------|
| ゲスト（非ログイン） | 未登録・未ログインユーザー。レシピ閲覧のみ可能。 |
| 一般ユーザー（会員） | ログイン済みユーザー。AIレシピ生成・調理ナビ・履歴管理など全一般機能を利用可能。 |
| アプリ管理者 | システム管理者。ユーザー管理・レシピ審査・サービス全体の運営を担う。管理画面へのログインが必須。 |

---
